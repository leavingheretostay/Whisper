# Whisper

> “A place where emotions float anonymously in a digital heaven.”

Whisper is a cinematic anonymous messaging platform designed to feel emotional, immersive, and beautifully minimal.

Users can create their own anonymous portal, share it with others, and receive thoughts, confessions, memories, and feelings — completely anonymously.

---

## ✨ Features

- Anonymous messaging
- Beautiful cosmic UI
- Glassmorphism design
- Real-time messaging
- Mobile-first responsive design
- Authentication system
- Supabase backend integration
- Smooth animations and transitions
- Ad-free experience
- Atmospheric aesthetic inspired by stars, dreams, and digital solitude

---

## 🌌 Design Philosophy

Whisper is not designed like a traditional social media app.

The goal is to create a peaceful digital space where:
- emotions feel weightless,
- conversations feel intimate,
- and anonymity feels beautiful instead of toxic.

The interface combines:
- soft gradients
- floating particles
- glowing ambient lighting
- dreamy typography
- cinematic motion

🚀 Getting Started

Clone the repository

git clone https://github.com/leavingheretostay/Whisper.git
cd Whisper

Install dependencies

npm install

Create environment variables

Create a ".env" file:

VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key

Run locally

npm run dev
